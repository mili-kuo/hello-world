## Getting Started

### Prerequisites

No dependencies required! This is a pure HTML and CSS project.

### Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/mili-kuo/hello-world.git
   ```

2. Open any `.html` file in your web browser:
   - Simply double-click any HTML file, or
   - Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (with http-server)
     npx http-server
     ```

3. Navigate to `http://localhost:8000` in your browser

## Files

- **index.html** - Main Hello World page with gradient background and centered card
- **index2.html through index11.html** - Additional page variants

## Customization

To customize the page:

1. Edit the HTML content in any `.html` file
2. Modify the CSS styles in the `<style>` section
3. Update colors, fonts, or layout as needed
4. Refresh your browser to see changes

### Example: Changing Colors

Modify the `background` property in the `body` style:
```css
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```
