Getting Started

Quick guide to get your Hello World project up and running

1

Clone the Repository

Get the project files from GitHub:

git clone https://github.com/mili-kuo/hello-world.git
2

Navigate to the Directory

Change into the project folder:

cd hello-world
3

Start a Local Server

Choose your preferred method to serve the files:

python -m http.server 8000

Or with Node.js: npx http-server

4

Open in Browser

Visit the local server in your web browser:

http://localhost:8000
5

Customize & Enjoy

Edit the HTML and CSS files to make it your own. Changes will appear when you refresh the browser.
